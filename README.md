# Find Me Recipe

Find Me Recipe is a provide users lots of recipes all around the world.
Lines of code Repo size

## Prerequisites
Python and Pip3
PostgresQL database 
Spoonacular API key

## Installation

```bash
pip3 -m env
source env/bin/activate
pip3 install -r requirements.txt
```
Database:
```bash
psql recipes
```
## Usage

```python3 server.py

```
